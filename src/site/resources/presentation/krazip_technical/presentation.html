Krazip Technical Overview

  • Krazip is not an official CruiseControl plug-in.
  • Krazip is an publisher plug-in
  • "Publishers are run after a build has completed. They will be run regardless of whether the build was successful or not." - CruiseControl
  • Krazip is actually extracts information from CruiseControl build log, processes it and sends them to configured IRC settings.
